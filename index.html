Карточки

Основные инструменты Semanticle – это статьи и карточки. Карточка – это автоматически выделенная из статьи информация. Содержит заголовок, картинку и описание. Карточку можно вставить на любую веб-страницу, а клик по ней откроет саму статью с полной информацией.

Пример отображения карточки

Это код для вставки карточки на клиентской стороне. Выбирая тип макета, вы управляете отображением карточки. 

Semanticle.card("57623e177b8b37f4220f1101", {
  "cardLayout": "side",
  "cardSideGalleryHeight": 142
}, document.getElementById("card-placeholder"));

Группы карточек

Semanticle умеет фильтровать карточки по тегам и категориям. Полученный результат можно вставить на страницу в виде карточек, открыть в модальном окне или получить в виде JSON для дальнейшей обработки. 

  Semanticle.find({
    "category":"врачи",
    "space":"57623dac7b8b37f4220f10fe",
    "skip":0,
    "limit":5
  }, {}, function(result) {
    console.log(result);
  }
